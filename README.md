# vim-caddyline

Vim syntax for [Caddy's](https://caddyserver.com/) [Caddyfile](https://caddyserver.com/docs/caddyfile).

## Installation
### Vundle
Put `Bundle 'joshglendenning/vim-caddyfile'` in your vimrc and do `:BundleInstall`.
